# SeamCarving
A C++ implement of the paper "Seam Carving for Content-Aware Image Resizing"

## Compile
```bash
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make
```

## Usage

```
# Example: seam_carving dolphin.jpg dolphin.png
seam_carving <input> <output>
```
